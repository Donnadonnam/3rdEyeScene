Detailed Description

A spin lock using atomic operations.

To ensure correct unlocking even on exception, the recommended usage is: 

      void ProtectedFunction(SpinLock lock)
      {
        lock.lock();
        try
        {
          // Your code here.
        }
        finally
        {
          lock.unlock();
        }
      }
