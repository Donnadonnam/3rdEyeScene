Detailed Description

Test scene serialisation and restoration

  • Create a TES server.
  • Connect the TesComponent to the server.
  • Create a scene with at least one of each type of shape.
  • Serialise the client scene.
  • Validate the scene.
  • Disconnect the server.
  • Restore the serialised scene.
  • Validate the scene.
  • Serialise the client scene again.
  • Compare serialised files.
