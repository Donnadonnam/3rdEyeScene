enum Tes.Main.NetworkThreadStatus
strong

Status of the NetworkThread.

Enumerator
None 

Invalid status.

Started 

Thread has started.

Not connecting yet.

Disconnected 

Thread has been disconnected.

It is important to still flush any pending messages once disconnected. Some sessions may be very short lived and all messages should still be processed and the initial connection handled.

Connecting 

Waiting for a connection.

Connected 

Currently connected.

Reconnecting 

Was disconnected, but waiting on a new connection.

As will Disconnected, all pending messages should be processed.

ConnectionFailed 

Failed to connect.
