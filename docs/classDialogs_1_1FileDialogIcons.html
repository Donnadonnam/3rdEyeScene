Detailed Description

Implements the FileIconSet and allows user configuration.

This script can be attached to an object to allow user configuration of the icon set.

Get the icon for entry

Parameters
entryThe file entry to match an icon for.
Returns
An icon to use for entry . May be the DefaultIcon.

Matching is made by Type, then SubType. SubType is only used where available. An icon with matching Type and no SubType matches any item of the type, regardless of its SubType.
