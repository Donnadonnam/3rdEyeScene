Detailed Description

The UI script for MessageBox.

The message box UI may be shared by different message boxes. However, we assume we can only display one at a time with any priority. To help with this, the UI maintains a stack of Owner items. Whenever an Owner is set, the previous one is pushed onto the stack. Once the UI closes, the previous Owner is restored and the UI calls MessageBox.ShowMessageOn() is called to update the UI to that owner. Setting a null Owner prevents this behaviour and clears the stack.
