Detailed Description

Parses command line arguments into a dictionary of arguments and values.

This class parses arguments of the forms:

  • –<name>=<value> into the Values dictionary of arg/value pairs. Leading '-' are dropped from the argument name.
  • –<arg> into the Opt array of strings. Leading '-' are dropped from the argument name.
  • Other arguments are put into Anonymous.
