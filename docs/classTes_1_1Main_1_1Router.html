Detailed Description

The Tes Router is the main control process for 3rd Eye Scene.

The Router manages the data active streaming thread and routes incoming messages to registered handlers.

Attempt to resolve a name for a routing ID based on the known ID set.

Parameters
idThe ID to resolve.
Returns
The name corresponding to id , or an empty string when unknown.

This method checks a set of known routing ID enumerations for a match and converts the entry to string when matched. The following enumerations are checked:
