Copies and maps a vertex from entry.Vertices unless it has already been mapped.

The mapped index is added to the input lists.

This is a helper function for mapping mesh data from the original data set into multiple Mesh objects to respect Unity's vertex count limit.

Each entry in indexMap corresponds to an element in entry.Vertices and is zero for unmapped vertices. For mapped indices, the value is the mapped index + 1.

A newly mapped vertex is added to vertices , normals , uvs , colours , skipping any null lists (except for vertices ).

Regardless of whether the vertex is already mapped, the vertex index is (re)added to indices .

Parameters
srcIndexThe vertex index in entry.Vertices
indexMapThe list of vertex mappings. A zero entry is unmapped.
entryThe mesh details.
indicesIndices for the new mesh component.
verticesVertices for the new mesh component.
normalsNormals for the new mesh component. May be null.
uvsUVs for the new mesh component. May be null.
coloursColours for the new mesh component. May be null.
