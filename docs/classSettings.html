Detailed Description

Base class for settings which can be displayed as part of the settings panel or another properties editor.

Derivations can expose any property for editing and serialisation to Unity's "Player Preferences" system. To do so, a properties must be correctly setup.

  1. Use only types convertable from string.
  2. Add System.ComponentModel.BrowsableAttribute to editable settings.
  3. In each property setter, invoke Notify(string) after setting the new value.

It is recommended that the properties are backed by Unity's PlayerPrefs or PlayerPrefsX.

Strictly speaking a class need not derive this one to expose browsable properties. The attributes and property notification are the critical components.
