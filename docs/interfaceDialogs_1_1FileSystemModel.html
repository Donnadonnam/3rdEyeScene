Detailed Description

This interface is used to traverse and iterate a file system.

The results may represent the real file system, or a virtual file system, as the use case dictates. The results may also be filtered or unfiltered.

Enumerate the children for a given entry.

Results are empty or null for childless entries.

Parameters
parentThe parent item to retrieve children for.
filterOptional regular expression filter matched against the entry Name field. A child entry must match the expression to be included in the result. May be null to return all children.
Returns
The child entries, a zero length array if there are no children (an empty directory) or null if there cannot be any children (files have no children).

The method must support a null entry parent type by listing the root items.
